# PulsePass - Health Scanning App

A React + TypeScript health scanning application built with Vite and Tailwind CSS.

## Project Structure

```
PULSEPASS/
├── src/
│   ├── components/          # React components
│   │   ├── SplashScreen.tsx
│   │   ├── WelcomeScreen.tsx
│   │   ├── LoginForm.tsx
│   │   ├── ScanInstruction.tsx
│   │   ├── ScanningScreen.tsx
│   │   ├── PostScanResult.tsx
│   │   ├── DashboardSimple.tsx
│   │   └── DeveloperSimple.tsx
│   ├── App.tsx              # Main app component
│   ├── main.tsx             # React entry point
│   └── index.css            # Global styles
├── index.html               # HTML entry point
├── package.json             # Project dependencies
├── tsconfig.json            # TypeScript config
├── vite.config.ts          # Vite config
├── tailwind.config.js      # Tailwind CSS config
└── postcss.config.js       # PostCSS config
```

## Getting Started

### Install Dependencies
\`\`\`bash
npm install
\`\`\`

### Development Server
\`\`\`bash
npm run dev
\`\`\`

### Build for Production
\`\`\`bash
npm run build
\`\`\`

### Preview Production Build
\`\`\`bash
npm run preview
\`\`\`
